设置用户名: git config --global user.name 'styleNB'

设置邮箱: git config --global user.emali 'yujiaming139147@vip.qq.com'

初始化文件: git init

查看状态： git status

添加暂存区： git add aa.html

提交： git commit -m '这里写描述'

删除： git rm aa.html

查看初始化信息： git config --list

复制项目： git clone 粘贴

将本地的仓库关联到github上： git remote add origin git@github com:leiphp/awesome-python3-webapp.git

上传到git仓库上： git push
